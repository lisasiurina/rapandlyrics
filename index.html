Context

This project analyzes 30 years of hip-hop lyrics using Python-driven data mining. By processing a corpus of 4,812 unique tracks, we mapped the shift from local community references to global luxury status symbols.

Key Focus: Quantitative analysis of automotive brand mentions and their correlation with commercial success in the Rap industry.

The Pipeline

01

Automated Scraping

I built scrapers to gather the raw dataset:

• Billboard Engine: Captures historical chart data using Python Requests.
• Genius Integrator: Fetches clean lyrics via API tokens, handling rate limits and metadata.

Python BeautifulSoup4 REST API
02

NLP Processing

Parsing slang and informal language required custom entity recognition logic. We mapped mentions like "Lambo" or "Caddy" to their official parent brands.

NLTK RegEx Text Processing
03

Data Aggregation

Using Pandas, we converted raw text into structured statistical data, allowing us to see trends over a 30-year horizon.

Pandas NumPy CSV/JSON
